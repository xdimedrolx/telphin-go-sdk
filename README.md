# Telphin SDK for Go

## Install

### Using go modules

In your go files, simply use:
```go
import "github.com/xdimedrolx/telphin-go-sdk/telphin"
```

### Using `$GOPATH`

```
go get github.com/xdimedrolx/telphin-go-sdk
```

automatically downloads the `master` branch in `$GOPATH/src`. Then in your
go files use:
```go
import "github.com/xdimedrolx/telphin-go-sdk/telphin"
```
